# レジスタンス：アヴァロン
[レジスタンス：アヴァロン](https://appmedia.jp/news/4770937)を遊ぶためだけのデータパックです
ただし、役職はアサシン、手下、家来、パーシヴァル、マーリンのみです。
## 動作環境
ver 1.16.4
前提データパックなし

# 使い方
1. 看板を用意します。
    1. ``loot give @s loot avaron:sign/..``の全ての看板を設置してください。
2. 参加者は全員アドベンチャーモードになります。
3. 参加者が7人以上いれば``次へ``の看板を押すことで始めることが出来ます。
